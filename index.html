About The Game

Square Dodger is a fast survival mini-game where each near miss becomes a point, milestone, or cosmetic unlock. It is designed to feel approachable in the first 10 seconds and more intense as the run builds.

Start a run, dodge the numbered bricks, build score, and earn credits for skins. Every 10 points celebrates progress, and every 50 points unlocks a power jump that smashes one brick for bonus score.
Trust note: the game runs fully in the browser, does not ask for sign-in, and keeps progress on the same device using local storage only.

How It Works

  • 1. Learn the controls fast Use keyboard or touch controls to move left, right, and jump. The game is tuned for quick pickup on both desktop and mobile.
  • 2. Survive to raise the pace Every dodge adds score. Difficulty ramps up gradually so new players can settle in before the pressure spikes.
  • 3. Keep progression between sessions Credits and unlocked skins stay saved locally, so returning players feel momentum instead of starting cold every time.
